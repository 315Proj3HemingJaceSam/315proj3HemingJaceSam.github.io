Name: Stock Retrieval Pro


Description: Financial stock quote retrival application with location sharing services incorporated
             Enables both dies-hard stock fans and novices to get a head up in the stock market

API: Yahoo Finance API queried through YQL (Yahoo Query Language)
     Google Map Search Box API (Google Map API)

Functionalities: Scrapes for financial quote from SP 500 companies and update them as you search through the drop down box
                 Automatically updates and queries from the 9 selected well-known companies and display them on the front page
		 Upon clicking the See Nearby Broker, button onclick leads it to getLocation() and pass the latitude and 
                 longitude to the Google Map API.
